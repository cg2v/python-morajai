## Blue Prince Mora Jai solver
python solver for the Mora Jai minigame in [Blue Prince](https://www.blueprincegame.com/)

Uses [python-astar](https://github.com/jrialland/python-astar)

Limitations (i.e. please use a known working version like https://joric.github.io/blueprince unless you are a python tinkerer)
* Not all colors are tested (red, blue, violet, white are not done)
* the heuristics are bad, so the path is not the shortest
